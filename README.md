# Agno Multi-Agent Data Science System

This project is a multi-agent system built with the Agno framework. It automates different stages of a standard machine learning pipeline—from data ingestion and exploratory data analysis (EDA) to feature engineering and model training—by delegating tasks to a team of specialized agents.

## How it Works

The core idea is to have separate agents handle specific parts of the data science workflow under a single "Agent OS" running on FastAPI. 

The system relies on a team of specialized agents:
- **Data Loader Agent:** Reads and samples CSV files while keeping context limits in check.
- **File Manager Agent:** Handles the project's filesystem and directory navigation.
- **Data Understanding Agent:** Uses Pandas to summarize data, check types, and run basic EDA operations (like `head()`, `describe()`, and `value_counts()`).
- **Visualization Agent:** Takes processed data and generates appropriate Matplotlib charts (histograms, scatter plots, etc.) and saves them to the disk.
- **Coding Agent:** Writes the actual Python code for data cleaning, modeling, and evaluation. It can also search DuckDuckGo for documentation.
- **Shell Agent:** A restricted agent that executes the generated Python scripts or installs dependencies.

All of these are orchestrated by a central **Data Science Team Leader** agent that guides the user step-by-step, asks for code review before execution, and stores important paths and context in a session state.

## Project Structure

- `app.py`: The main Agent OS application and agent definitions.
- `main.py`: A basic entry point script.
- `data/`: Where datasets belong (e.g., `car_details.csv`).
- `plots/`: Output directory for generated matplotlib charts.
- `src/`: Destination for the Python scripts generated by the coding agent.
- `models/`: Destination for saved joblib models.
- `reports/`: Output directory for generated markdown reports.

## Getting Started

### Requirements
- Python 3.10 or higher
- [uv](https://github.com/astral-sh/uv) (recommended for package management)
- An OpenAI API key

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/HarshDaulatani/Agno-Multi-Agent-Data-Science-System.git
   cd Agno-Multi-Agent-Data-Science-System
   ```

2. Install dependencies:
   ```bash
   uv sync
   ```

3. Set up your environment variables. Create a `.env` file in the root directory:
   ```env
   OPENAI_API_KEY=your_api_key_here
   ```

4. Start the server:
   ```bash
   python app.py
   ```
